配置网络和ssh

这里设置网络和ssh方便去下载资源和远程连接

连接网络

如果你使用的是有线网络并且网络还有DHCP的话那么网络部分就非常的简单好配置了.直接检查一下是否可以联通外网就可以了。如果是WiFi可以通过桌面的网络图标进行连接WiFi，之后测试是否可以联通外网。 

ping -c4 gentoo.org
其他的连接方式可以参考ArchWiki。

配置SSH

我们首先要给root用户设置一个密码作为ssh登录时连接使用 
passwd
默认情况下Ubuntu Livecd是没有openssh服务的我们需要先安装一个openssh-server的包 
apt-get update && apt-get install openssh-server
然后启动ssh服务 
systemctl start sshd
